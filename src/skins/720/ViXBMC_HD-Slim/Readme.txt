To enable the 2nd infobar:
Select menu
Select setup
select system
Select Customize
Scroll down to show second infobar and change from eventview to 2nd infobar


To adjust the fontsize for the quickepg do the following:
Press direction key left/right
Select menu
Select Items per page and adjust to 2
Select Fontsize (relative to skin size) adjust this up/down to your liking.

To adjust how many items per page for the Channelselection do the following so it does not appear squashed:
Press direction key up/down
Select menu
Select settings
Select Number of items and change this to 5 as more then 5 will make the list appear squashed
